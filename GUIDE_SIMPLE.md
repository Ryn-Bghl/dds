# 📝 GUIDE D'ÉDITION DU SITE - POUR NON-DÉVELOPPEURS

## Bienvenue! 👋

Ce guide vous explique comment modifier le site **sans avoir besoin de coder**.

Vous avez juste besoin de:
- ✅ Un navigateur (Chrome, Firefox, Safari, Edge)
- ✅ Les identifiants de connexion
- ✅ 5 minutes! 😊

---

## 🚀 ÉTAPE 1: Accéder à l'Administration

### Si le site est en local (développement):
```
1. Ouvrez votre navigateur
2. Allez à: http://localhost:5173/login
3. Entrez vos identifiants
4. Cliquez "Se connecter"
```

### Si le site est en production (Vercel):
```
1. Allez à: https://donsduson.vercel.app/login
2. Entrez vos identifiants
3. Cliquez "Se connecter"
```

---

## ✏️ ÉTAPE 2: Activer le Mode Édition

```
1. Une fois connecté, vous voyez le "Tableau de Bord"
2. En haut à droite, trouvez le bouton:
   ➡️ "Activer l'Édition Directe" (ou "Mode Édition Actif")
3. Cliquez dessus
```

**Vous verrez:**
- Les textes ont maintenant des petites icônes ✏️ quand vous les survolez
- Tout devient éditable!

---

## 🎨 ÉTAPE 3: Modifier le Contenu

### Pour modifier n'importe quel texte:

```
1. Allez sur la page que vous voulez modifier
   Exemples: Accueil, Association, Événements, etc.

2. Cherchez le texte à modifier
   Exemples: Le titre, une description, etc.

3. Survolez le texte avec votre souris
   ➡️ Un petit crayon ✏️ apparaît

4. Cliquez sur le texte
   ➡️ Une boîte d'édition s'affiche

5. Modifiez le texte comme vous le souhaitez
   (Effacez, écrivez, copiez-collez, etc.)

6. Cliquez sur ✓ (le bouton vert avec une coche)
   ➡️ C'est enregistré temporairement!
```

**Si vous vous trompez:**
- Cliquez sur ✗ (le bouton rouge avec un X)
- Les changements seront annulés

---

## 💾 ÉTAPE 4: Sauvegarder les Modifications

### Après avoir modifié tous les textes que vous voulez:

```
1. En haut à droite, trouvez le bouton:
   ➡️ "Enregistrer les modifications" (en vert)

2. Cliquez dessus
   ➡️ "Modifications enregistrées avec succès!" ✅

IMPORTANT: Ne fermez pas le navigateur avant de cliquer!
```

## 📋 EXEMPLES CONCRETS

### Exemple 1: Modifier le titre de la page d'accueil

```
1. Mode Édition Activé ✓
2. Allez à "Accueil" (menu)
3. Cherchez le grand titre en haut
4. Survolez → Crayon ✏️ apparaît
5. Cliquez
6. Modifiez le texte (ex: "Donnons du son..." → "Notre Association...")
7. Cliquez ✓
8. Cliquez "Enregistrer les modifications" en haut à droite

```

```
4. Cliquez sur un des chiffres
5. Modifiez le nombre (ex: 4 → 5)
6. Cliquez ✓
7. Vous pouvez modifier les autres chiffres aussi
8. Cliquez "Enregistrer les modifications"
9. ✅ C'est publié!
```

### Exemple 3: Modifier une description

```
1. Mode Édition Activé ✓
2. Allez à "Association" 
3. Cherchez le texte que vous voulez modifier
4. Survolez → Crayon ✏️
5. Cliquez
6. Modifiez le texte
7. Cliquez ✓
8. Enregistrez les modifications
9. ✅ Fait!
```

## ⚠️ POINTS IMPORTANTS

### À FAIRE ✅:
- ✅ Activer le Mode Édition avant de modifier
- ✅ Cliquer sur ✓ après chaque modification
- ✅ Cliquer "Enregistrer les modifications" quand vous avez fini
- ✅ Attendre que le message "Enregistré!" s'affiche

### À NE PAS FAIRE ❌:
- ❌ Ne pas fermer le navigateur sans enregistrer

## 🆘 Problèmes?

### Le mode édition n'apparaît pas:
```
1. Vérifiez que vous êtes connecté ✓
2. Rafraîchissez la page (Ctrl+R ou Cmd+R)
3. Réessayez
```

### Les modifications ne s'enregistrent pas:
```
1. Vérifiez que vous avez cliqué ✓ (coche verte)
2. Vérifiez que vous avez cliqué "Enregistrer les modifications"
3. Attendre que le message vert s'affiche
4. Rafraîchissez la page pour vérifier
```

### Un crayon ✏️ n'apparaît pas sur un texte:
```
1. Survolez très lentement
2. Attendre 1-2 secondes
3. Si ça ne marche pas, c'est peut-être un élément qui ne peut pas être édité
4. Contactez le développeur
```

### Vous avez fermé sans enregistrer:
```
1. ❌ Les modifications non-enregistrées sont perdues
2. Il faudra les refaire
3. N'oubliez pas "Enregistrer les modifications" à la fin!
```
